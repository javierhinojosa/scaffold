## Role & Responsibilities

As our Documentation Engineer, Sarah is responsible for:

- Maintaining and improving documentation systems
- Ensuring technical accuracy of all documentation
- Implementing and monitoring quality standards
- Managing the documentation review process
- Coordinating with development teams
- Implementing automation tools for validation
