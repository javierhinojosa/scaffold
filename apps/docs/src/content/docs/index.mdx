## Welcome to SFH

This monorepo contains all the code and resources for the SFH project. Our repository is organized using modern development practices and tools to ensure maintainability, scalability, and developer productivity.

### Repository Structure

The monorepo is organized into two main directories:

- `apps/*` - Contains all our applications
  - `admin` - Admin dashboard application
  - `docs` - This documentation site
  
- `packages/*` - Contains shared libraries and utilities
  - `testing` - Common testing utilities and configurations

### Key Features

- **Turborepo** for build system orchestration
- **pnpm** for fast, disk-space efficient package management
- **TypeScript** for type-safe development
- **Automated testing** with Vitest and Playwright
- **Modern tooling** including ESLint, Prettier, and more

### Getting Started

Check out our [Getting Started guide](/guides/getting-started/) to set up your development environment and make your first contribution.

### Tech Stack

- Node.js >=18
- pnpm 8.15.4
- TypeScript 5.7.3
- Astro for web applications
- Vitest for testing
