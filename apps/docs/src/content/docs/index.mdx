Scaffold is a carefully crafted template monorepo that combines modern development tools, practices, and architectures to provide a robust foundation for any organization's development needs. Whether you're building web applications, AI services, documentation sites, or complex enterprise systems, this template provides the structure and tooling to accomplish it all.

### Repository Structure

The monorepo demonstrates a scalable organization pattern with:

- `apps/*` - Contains example applications showcasing different capabilities
  - `admin` - Modern admin dashboard application
  - `docs` - Documentation site with best practices
  - `crewai` - AI-powered automation and workflows
- `packages/*` - Shared libraries and utilities
  - `testing` - Common testing utilities and configurations

### Key Features

- **Turborepo** for efficient build system orchestration and task running
- **pnpm** for fast, disk-space efficient package management
- **TypeScript** for type-safe development across all projects
- **Comprehensive testing** infrastructure with Vitest and Playwright
- **AI Integration** capabilities through CrewAI
- **Modern tooling** including ESLint, Prettier, and automated workflows
- **Documentation** infrastructure for maintaining clear project knowledge
